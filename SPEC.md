# AI习惯锻造 — AI Habit Forge

## Concept
Pure frontend habit tracker with AI coaching. Atomic Habits methodology: start tiny, track streaks, get AI encouragement and recommendations.

## Features
- Add habits with frequency (daily/weekly)
- Track daily (tap to complete)
- Streak counter + longest streak
- AI coaching: personalized tips via template matching
- Weekly summary
- Clean minimalist UI
- localStorage persistence

## Specs
- Single HTML file
- Pure frontend
- Template-based AI coaching (no API key needed)
